# Claused / Inbora

## Stack
- React (Create React App)
- Deployed on Vercel (NOT Netlify)
- Live at inbora.co
- GitHub: github.com/loxandlottie-prog/claused

## Deploy
Run `ship` to push (git add . && git commit -m "update" && git push)
Vercel auto-deploys from main branch.

## Auth
Google Cloud OAuth — redirect URI uses window.location.origin dynamically

## Key files
- src/tabs/ — main tab components
- api/gmail/ — Gmail integration
- src/components/BrandCard.js — brand card UI
